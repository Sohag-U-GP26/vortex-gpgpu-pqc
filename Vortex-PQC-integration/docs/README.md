# Documentation — Phase 2 Artifacts

> Part of [Phase 2: Vortex GPGPU Integration](../README.md) — test outputs, performance data, and architecture documentation.

This directory contains test outputs, performance data, and architecture documentation for Kyber polynomial multiplication on the Vortex GPGPU.

## Files

| File | Description |
|------|-------------|
| `architecture_and_implementation.md` | Detailed architecture and implementation documentation |
| `kyber_ntt_architecture.md` | ASCII flow diagram of the NTT-based polynomial multiplication on the GPGPU |
| `performance.md` | Detailed performance analysis |
| `performance_comparison.md` | Performance comparison between polymul and kyber kernels (instructions, cycles, IPC) |
| `performance_summary.md` | Structured performance summary with test configuration and results |
| `results.md` | Test results |
| `running.md` | Build and run instructions |
| `test_output.log` | Full raw test output from RTL (Verilator) simulation — all 7 test cases |
| `test_output.md` | Formatted markdown version of the test output |
